Weather API

api.open-meteo.com + geocoding
Enter a city name above to fetch live weather.
// Fetch weather data const geo = await fetch( `https://geocoding-api.open-meteo.com/v1/search?name=${city}` ); const weather = await fetch( `https://api.open-meteo.com/v1/forecast?...` );
😂

Joke API

v2.jokeapi.dev
Click below to fetch a random joke!
// Fetch a random joke const res = await fetch( 'https://v2.jokeapi.dev/joke/Any' ); const data = await res.json(); if(data.type === 'twopart') { show(data.setup, data.delivery); } else { show(data.joke); }

Advice Slip API

api.adviceslip.com
Click below for a piece of advice!
// Advice Slip API const res = await fetch( 'https://api.adviceslip.com/advice' ); const { slip } = await res.json(); display(slip.advice);
